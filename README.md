Unity Heart Scan Render
=====

This is a demo of the pipeline from https://github.com/Thomaslund1/Heart-Scan-Pipeline.git

This is the Unity build for Linux, but should be compatible with other systems. 

To extract the files navigate to the data folder and run 

  unzip UnityZip.zip

or equivalent in your os

To run the demo, run the text.exe.x86_64 file
